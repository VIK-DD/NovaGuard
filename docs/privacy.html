Privacy Policy

This Privacy Policy explains what information NovaGuard may process, why it is used and how it is handled. NovaGuard is a self-hosted Discord bot developed by VIK & CloudMedia.

1. Information We May Process

NovaGuard may process Discord user IDs, guild IDs, channel IDs, role IDs, server configuration data, moderation or feature state, and other metadata required to operate the bot's features.

2. Purpose of Processing

Data is used only to provide requested bot features, preserve configuration, keep feature state, generate embeds, support moderation tools and maintain service stability.

3. Storage

NovaGuard stores selected bot data in self-hosted local storage, including SQLite databases and JSON files, depending on the active feature.

4. Third-Party Services

NovaGuard may use third-party services such as Discord, GitHub and optional AI providers. When those integrations are enabled, relevant data may pass through those services under their own policies.

5. Data Sharing

NovaGuard does not sell personal data. Information is only shared when required for core bot functionality or optional integrations explicitly used by the operator.

6. Retention

Data may be retained as long as needed for bot operation, server continuity, backups and configuration persistence, unless manually removed by the operator.

7. Security

Reasonable efforts are made to protect stored data, but no method of transmission or storage is completely secure. Server owners should use safe secrets handling and secure hosting practices.

8. Contact

For privacy-related questions, contact the operator through the NovaGuard repository or the NovaGuard GitHub profile.
