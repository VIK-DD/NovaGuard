# NovaGuard Coming Soon

Clean static export for the NovaGuard coming soon page.

## Run locally

From the bot folder:

```bash
cd website-3
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy

Upload everything inside this folder to your hosting provider:

```text
index.html
assets/
_redirects
README.md
```
